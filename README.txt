Requirements: 

- Java 1.5
- Log4j (http://logging.apache.org) required in order to compile Open Chord
  from sources. 
- Ant (recommended, http://ant.apache.org) to compile Open Chord.


Available files: 

open-chord_1.x.x.zip: Contains the complete Open Chord version 1.x.x with src,
documentation, and compiled files.


Installation:

Just unzip the desired zip file to any directory.
In that directory you will find the Open Chord directory structure.

The manual of Open Chord is located in directory docs/manual as pdf file.
To compile/use Open Chord follow the instructions in that manual.
